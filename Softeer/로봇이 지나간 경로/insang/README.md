# [Backjoon] 로봇이 지나간 경로

### 접근법

-   시뮬레이션 문제
-   주어진 입력은 로봇의 이동 경로이므로, 경로가 끊기지 않고 하나로 이어져 있음.
-   따라서, 경로의 시작 지점이나 끝 지점만 찾으면 풀이가 가능.

-   search 함수

    -   모든 좌표에서 로봇의 이동 경로를 탐색

-   check 함수

    -   해당 좌표가 경로의 시작 지점인지, 끝 지점인지 구분하기 위한 함수
    -   특정 좌표에서 4방 탐색을 진행하여 진행 가능한 방향이 하나뿐이라면 시작 or 끝 지점

-   go 함수
    -   이동 경로의 시작 or 끝 지점을 찾은 뒤, 로봇의 커맨드 출력을 위한 함수
    -   while문 (로봇이 이동한 길이 < 전체 경로의 길이) 반복
