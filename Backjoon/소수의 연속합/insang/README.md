# [Backjoon] 소수의 연속합

### 접근법

-   소수 판별 알고리즘 (에라토스 테네스의 체)

    -   완전 탐색을 이용한 소수 판별 -> O(N^2) : 시간 초과
    -   제곱근 소수 판별 -> O(N루트N) : 제한 시간 내 통과
    -   에라토스테네스의 체 -> O(N log log N) : 제한 시간 내 통과
    -   시간 복잡도가 가장 낮은 에라토스테네스의 체 알고리즘 사용

-   연속된 소수의 합으로 나타낼 수 있는 경우의 수
    -   N 이하의 소수를 담을 리스트 선언
    -   해당 소수들의 누적합 리스트 선언
    -   투 포인터 + 누적합 + 슬라이딩 윈도우 기법을 활용해 문제 풀이
